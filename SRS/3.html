3.3 Software Interfaces

The backend will be built with Django and PostgreSQL will be the server hosted locally that connects to the backend with Psycopg2 as the database adapter. Django templates will serve as the frontend for the web platform. Django REST API that sits outsite the backend system will help connect to the Mobile Platform front end built with React Native that also sits outsite the system. Web Crawlers will be made with Beautiful Soup and Selenuim and connected to the backend via their api. The data storage for scraped data by the crawlers are to be determined. Search indexing system will be implemented via Solr(a Lucene based system) that will similarly interact with the database via an api.


3.4 Communications Interfaces

The communicaiton between backend and multiple frontends are to be carried out following the REST standard. Communication methods for other systems are to be determined.
