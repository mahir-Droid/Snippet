4.4 Search Multple Cluster: R4


4.4.1 Description and Priority

User logged in, in their view cluster page, selects multiple cluster(s) and inputs text in the search box to search data in the selected search clusters corresponding to their text. The user is then redirected the search result page.


4.4.2 Stimulus/Response Sequence

Actor action System Response
1. A logged in user in their view cluster page selects multiple cluster and inputs text into the search box. 2. System searches the text in the data of the muliple cluster selected and redirects to the view results page.


4.4.3 Functional Requirements

User must already have at least one cluster created and saved in their profile to execute this function.
