4.2 Signup: R2


4.2.1 Description and Priority

Any new user wishing to adopt our application will visit the website or use the app to register themselves as a User by inputting their username email and password into the system.


4.2.2 Stimulus/Response Sequence

Actor action System Response
1. A new user visits the website signup page. 2. Prompts a signup screen asking for their credentials: username, email, password and confirm password.
3. User inputs their credentials and clicks on signup button 4. Signup credentials are verified, user is logged in automatically and prompted with their personal view cluster screen.


4.2.3 Functional Requirements

The user must input a valid email and passwords fields that match. If there is an invalid email input and/or mismatching passwords the user is prompted back to the signup page with appropriate error message(s).
