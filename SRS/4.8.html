4.8 Voice Input Search: R8


4.8.1 Description and Priority

A user in the mobile platform who are in the view cluster page has the option to use voice command to input search text.


4.8.2 Stimulus/Response Sequence

Actor action System Response
1. User clicks mic button. 2. System starts recording voice from device microphone.
3. User speaks the search text and clicks end recording. 4. The recording is translated to text and sent to search from cluster.


4.8.3 Functional Requirements

User must be in the mobile platfrom for this feature to work. Furthermore, microphone access must be granted by the user to the app.
